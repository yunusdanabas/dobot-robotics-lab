# Lab 03: Path Planning

Prepared for **ME403 Introduction to Robotics** by **Yunus Emre Danabas**.

Implement path planners for the printed workspace maps described in the lab handout.

## Files

| File | Purpose |
|------|---------|

Python starter files for this lab will be added in a future release. Follow your instructor's folder assignment for submission.

## Preparation

Print the maps from the handout and place them on the table in front of the robot. Measure obstacle sizes and positions before coding.

## Tasks (summary)

1. Implement **Artificial Potential Fields** and one **probabilistic method** (PRM or RRT) for maps 1–2.
2. Test three start/goal pairs per map; plot and compare path length.
3. *(Bonus)* Apply a planner of your choice to maps 3–4.
