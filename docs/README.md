# Docs

Student-facing guides for setup, simulation, robots, and troubleshooting. Paths in the list below are under this `docs/` folder unless noted.

**Scope:** Robot and workflow pages here are **generic** (no numbered “Lab N” in those files). Exercise-specific `utils.py` wiring and folder names are documented in `[student_api.md](student_api.md)` (and optional copies beside each exercise).

Recommended reading order:

1. `[../README.md](../README.md)` (package root — canonical start)
2. `[setup.md](setup.md)`
3. `[student_api.md](student_api.md)` (exercise helpers, env vars, kinematics — keep open while coding)
4. `[simulation.md](simulation.md)`
5. `[magician_setup.md](magician_setup.md)` or `[mg400_setup.md](mg400_setup.md)` (hardware tracks)
6. `[troubleshooting.md](troubleshooting.md)`
7. `student_simulation_pack_guide.pdf` (release explainer)
8. `magician_calibration_and_setup_guide.pdf` (real Magician calibration workflow)
